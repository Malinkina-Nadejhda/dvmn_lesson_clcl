# Создание коротких ссылок и подсчет кликов.

Программа создает короткую ссылку с помощью сервиса [clc.li](https://clc.li/). По короткой ссылке программа подсчитает количество переходов по ней.

## Как установить

- Должен быть установлен Python3+. 
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements.txt
```

## Установка и запуск

Клонируйте репозиторий. Получите токен на сайте [clc.li](https://clc.li/). Создайте `.env` файл в корне проекта и скопируйте содержимое из `.env.example` в `.env`. Вставьте ваш реальный токен в файл `.env`.  Запустите программу `python lesson2_clc.py` и введите ссылку. Длинная ссылка будет сокращена, короткая покажет количество переходов по ней за все время.

## Пример
**Ввод:**

Введите URL: https://liteka.ru/english/library/6678-harry-potter-and-the-sorcerer-s-stone#3

**Вывод:**

Короткая ссылка: [clc.li/dPvMA](https://clc.li/dPvMA)
