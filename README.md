# Создание коротких ссылок и подсчет кликов.

Программа создает короткую ссылку с помощью сервиса [clc.li](https://clc.li/). По короткой ссылке программа подсчитает количество переходов по ней.

## Как установить

Должен быть установлен Python3. Используйте **pip** для установки зависимостей

pip install -r requirements.txt

## Как пользоваться

Получите токен на сайте [clc.li](https://clc.li/). Добавьте его в **.env** файл. Запустите программу и введите ссылку. Длинная ссылка будет сокращена, короткая покажет количество переходов по ней за все время.
